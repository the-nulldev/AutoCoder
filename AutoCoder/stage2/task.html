Description

In this stage, you will develop a basic workflow file to recap the basics. The action runs on push events and prints "Hello, world!"

Objectives

In this stage, create a main.yml inside the .github/workflows/ directory with the following properties:

  • It is triggered on push events;
  • It has one job named say_hello with two steps;
  • The first step uses the actions/checkout action to checkout the repository;
  • The second step runs a shell command to display "Hello, world" with the echo command;
